# rapid
rapid bloilerplate code generation
## Usage

<pre>
python main.html language > file_name
</pre>
###example

<pre>
python main.html html > index.html
</pre>

##Currently Supported
* C
<pre>
python main.html c > main.c
</pre>
* C#
<pre>
python main.html csharp > Hello.cs
</pre>
* Java
<pre>
python main.html java > Hello.java
</pre>
* HTML
<pre>
python main.html html > index.html
</pre>
